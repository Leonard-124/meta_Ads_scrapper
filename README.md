
This is minimal production-ready Meta Ads Library scraper (~100 lines).

Features:
- Search by keyword, Facebook page URL, or page slug
- Collect structured ad data
- Handle pagination, retries, and 429 backoff
- Export to CSV or JSON
- Embedded access token (as requested)This is minimal production-ready Meta Ads Library scraper (~100 lines).

How to work with it, use your Meta Ads API access token: replace it at this part:

ACCESS_TOKEN = "PASTE META ACCESS TOKEN"

Ensure that your facebook developers account has the necessary authorizations to access your App and use its access token

Get the access token from here:  https://www.facebook.com/ads/library/api/     click access API

on your terminal run python web_scrapper.py

to clone this repo run git clone
